This is a website project for System Integration and Architecture Laboratory course. 
In partnership of Street Furpaws, a local, non-goverment organization, and independent rescuer from City of Sta. Rosa, Laguna

The website includes all the existing information about Street Furpaws; blog, history, contact information, etc. as well as a custom 
adoption, donation, and volunteer system that allows users to sign in or sign up in order to submit a form to adopt or volunteer.
